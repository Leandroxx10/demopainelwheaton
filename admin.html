Configurar Limites por Tipo de Processo

Os limites são definidos automaticamente pelo tipo de processo da máquina.
SG, DG, TG, QG, PS, PD, AMOSTRA - cada um tem valores predefinidos.

Como funciona o novo sistema

1. Selecione uma máquina na lista abaixo
2. O sistema mostrará o tipo de processo atual (se existir)
3. Se não existir, selecione um tipo de processo na lista
4. Ao selecionar um processo, ele será salvo automaticamente
5. Os limites serão aplicados automaticamente

Para alterar os valores dos processos, entre em contato com o administrador

Como Funciona

Novo Sistema: Agora você pode atribuir prefixos detalhados às máquinas usando a seção "Gerenciar Máquinas".

Para atribuir um prefixo:

  1. Vá para "Gerenciar Máquinas"
  2. Clique no botão "Prefixos" da máquina desejada
  3. Selecione um prefixo da lista de prefixos detalhados
  4. Clique em "Salvar"

Esta seção agora é apenas para visualização. Use "Prefixos Detalhados" para criar novos prefixos.
